## 🖨️ How to Print

### Method 1: Direct Browser Print
1. Open the HTML file in any web browser (Chrome recommended)
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Settings:
   - **Paper size:** A4
   - **Margins:** Default
   - **Scale:** 100%
   - **Background graphics:** ON (important for colors)
4. Print or Save as PDF

### Method 2: Save as PDF First
1. Open in browser
2. Print to PDF
3. Use the PDF for printing or sharing

## 🎤 Common Questions to Prepare For

1. **"What's the commission rate?"**
   - Explain the percentage (flexible based on negotiation)
   - Emphasize no upfront costs or monthly fees

2. **"Who pays for the camera?"**
   - Playmaker covers 100% of installation and maintenance
   - Zero cost to the partner

3. **"How do I get paid?"**
   - Automatic bank transfers
   - Choose frequency: daily, weekly, or monthly
   - Show admin app payment tracking feature

4. **"What if I want to stop?"**
   - Explain the 14-day notice period
   - No penalties or fees
   - Camera remains Playmaker property

5. **"Can I still take walk-in bookings?"**
   - Yes, admin app allows real-time availability updates
   - Block times for private bookings anytime
