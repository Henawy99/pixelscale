# 🚀 Quick Start: Partner Presentation

## Open & Print in 3 Steps

### Step 1: Open the File
```bash
# Navigate to the file location
cd /Users/youssefelhenawy/Documents/playmakerstart/assets/contracts

# Open English version
open playmaker_partner_presentation.html

# Open Arabic version
open playmaker_partner_presentation_ar.html
```

Or simply **double-click** the file in Finder:
- `playmaker_partner_presentation.html` (English)
- `playmaker_partner_presentation_ar.html` (Arabic)

### Step 2: Customize Contact Info (Page 5)
Before printing, update these on the last page:
- **Phone Number:** Add your actual phone number
- **WhatsApp:** Add your WhatsApp number
- Email is already set: `partners@playmaker.com`

To edit: Right-click the HTML file → Open With → TextEdit or any code editor.

### Step 3: Print or Save as PDF

**In Chrome/Safari:**
1. Press `Cmd + P` (Mac) or `Ctrl + P` (Windows)
2. **Settings:**
   - Paper: A4
   - Margins: Default
   - Scale: 100%
   - ✅ **Background graphics: ON** (important!)
3. Click "Print" or "Save as PDF"
